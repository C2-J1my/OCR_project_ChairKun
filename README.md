# OCR 项目前端优化说明

## 使用说明

- 将本分支修改的index.html替换主分支的index。html
- 主分支新建static文件夹，加入本分支的favicon.ico（华工校徽文件）

## 主要修改内容

本次修改主要聚焦于 `index.html` 的展示增强和端侧预处理功能实现，具体包括以下几个方面：



 **UI/UX 优化**

- 增加响应式设计，适配不同屏幕尺寸
- 美化上传区域、按钮、结果展示等组件样式
- 添加加载动画效果，提升用户等待体验
- 新增图片预览功能，上传后可直接查看图片

 **前端预处理功能 (S1 分区)**

- 实现基于 `<canvas>` 的端侧轻量预处理功能
- 支持图片缩放（最大宽度限制为 800px）和灰度化处理
- 添加预处理开关，可手动切换是否启用前端预处理
- 实时显示前端预处理耗时和处理后的图片尺寸

**结果与时间展示增强**

- 细化时间统计展示，区分：


- 前端预处理时间
- 后端预处理时间
- OCR 识别时间
- 总耗时（前端 + 后端）
- 优化识别结果展示区域，提升可读性
- 增加错误信息展示区域，统一错误处理流程

 **用户体验提升**

- 显示上传文件的名称和大小信息
- 优化按钮交互反馈
- 增加操作指引文本
- 添加网站图标（学校校徽 `static/favicon.ico`）
